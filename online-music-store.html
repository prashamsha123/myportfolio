Online Music Store

A responsive music store website with modern UI and interactive features.

Project Overview

This project is a responsive online music store built with HTML, CSS, and JavaScript. It features a modern interface, dynamic search functionality, and a user-friendly layout.

Features

  • Dynamic search bar for quick music searches
  • Interactive product cards with hover effects
  • Responsive design for all devices

Technologies Used

  • HTML, CSS, JavaScript
  • Responsive design with Flexbox and Grid
  • Interactive elements with JavaScript
