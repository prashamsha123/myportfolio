Novel Reading App

A user-friendly application for an enhanced novel reading experience.

Project Overview

This project is a mobile and web application designed to provide an immersive novel reading experience. The app features a clean and intuitive interface, bookmark functionality, and dark mode for a comfortable reading experience.

Features

  • Bookmarking to save your reading progress
  • Dark mode for low-light environments
  • Responsive design for mobile and desktop devices

Technologies Used

  • Figma for UI/UX Design
  • HTML, CSS, JavaScript for the web version
  • React Native for the mobile app
